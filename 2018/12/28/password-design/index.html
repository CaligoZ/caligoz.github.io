产品设计：密码设置要求

设置密码要求：

在“密码要求”部分的最小密码长度字段中，设置用户密码的最小长度。缺省的最小密码长度为 6 个字符。在密码中允许相同的相邻字符数字段中，设置密码中允许的相同相邻字符的数量。选中密码可包含登录标识复选框，以允许用户在密码中使用登录标识。

设置要求的密码字符：

在“要求的密码字符”部分中，设置以下规范：必须包含大写字符：如果选中此复选框，那么密码必须至少包含一个大写字符。必须包含小写字符：如果选中此复选框，那么密码必须至少包含一个小写字符。必须包含数字：如果选中此复选框，那么密码必须至少包含一个数字字符。数字字符为：1 2 3 4 5 6 7 8 9 0。必须包含特殊字符（!、@、# 等）- 如果选中此复选框，那么密码必须至少包含一个特殊字符。支持的特殊字符为：! @ # $ % ^ & * ( ) - _ = + \ | [ ] { } ; : / ? . > <

允许放置密码字符：

在允许放置密码字符字段中，设置以下规范：第一个字符可以为数字：如果选中此复选框，那么密码的首个字符可以是数字。最后一个字符可以是数字 - 如果选中此复选框，那么密码的最后一个字符可以是数字。数字字符包括：1、2、3、4、5、6、7、8、9、0。第一个字符可以为特殊字符：如果选中此复选框，那么密码的首个字符可以是特殊字符。 支持的特殊字符为：! @ # $ % ^ & ( ) - _ = + \ | [ ] { } ; : / ? . > < . 最后字符可以为特殊字符：如果选中此复选框，那么密码的最后一个字符可以是特殊字符。 支持的特殊字符为：! @ # $ % ^ & ( ) - _ = + \ | [ ] { } ; : / ? . > < .

创建排除的密码列表：

在“排除的密码列表”部分中，单击新建行。在密码字段中，输入要在系统中禁止使用的密码

删除排除的密码：

在“排除的密码列表”部分中，单击要删除的已排除密码旁边的标记为待删除。

参考：https://www.ibm.com/support/knowledgecenter/zh/SSFGJ4_7.6.0/com.ibm.mbs.doc/mbs_common/t_set_password_reqs.html
